# 落地页工单制作 Skill

## 功能概述

基于 AI 视觉能力，自动分析竞品电商详情页图片的结构和角色，结合我方产品卖点，生成符合电商转化逻辑的设计工单。

**核心能力**：
- 识别每张图在详情页中的角色（首屏/痛点/功效/信任/对比等）
- 针对不同角色输出对应风格的文案（避免重复堆砌卖点）
- 使用 10 年电商专家的运营思维生成高转化文案
- 输出标准化 Word 工单，可直接交付设计师

---

## 使用流程

### Step 1: 环境准备

首次使用时，在项目目录下执行：

```powershell
// turbo
pip install google-generativeai python-docx Pillow python-dotenv
```

### Step 2: 配置 API Key

在项目目录创建 `.env` 文件：

```
GOOGLE_API_KEY=你的API密钥
```

### Step 3: 询问产品信息

**开始任何产品的落地页制作前，必须询问用户以下信息**：

1. **产品名称**：例如"雪叶红口服液"
2. **文案方向**：例如"孕妇补铁"、"气血虚补血"、"宫寒痛经"
3. **产品重点**（固定属性，无法改变的）：
   - 功能性：如"补血养血"、"改善睡眠"
   - 原料成分：如"阿胶"、"红参"、"铁元素"
   - 权威背书：如"国食健字号"、"百年老字号"
   - 目标人群：如"孕妇"、"职场女性"、"中老年"
4. **竞品素材位置**：包含主图和详情页切图的文件夹路径

### Step 4: 准备竞品素材

确保文件夹结构如下：
```
竞品文件夹/
├── 主图/           # 主图图片（5张左右）
└── 详情页（切图）/  # 已切好的详情页图片（按顺序命名）
```

### Step 5: 创建产品卖点文件

根据用户提供的产品重点，创建卖点文件 `产品名称卖点.txt`，格式如下：

```
【产品名称】雪叶红口服液

【核心功效】
- 补血养血：富含铁元素和阿胶，有效改善贫血
- 改善气色：14天改善肤色暗沉
- 增强免疫：多种营养素综合作用

【权威背书】
- 国食健字号认证
- 百年老字号品牌
- SGS检测认证

【目标人群】
- 孕妇/备孕女性
- 气血不足的职场女性
- 月经量大的女性

【产品规格】
- 规格：10ml×30支
- 服用方法：每日2次，每次1支
```

### Step 6: 执行生成

运行主程序：

```powershell
python scripts/main.py
```

程序会：
1. 先处理主图（生成5张主图的文案）
2. 再处理详情页切图（识别每张图的角色，生成对应文案）
3. 输出 Word 工单到 `output/` 目录

## 电商运营思维说明

### 详情页图片角色识别

AI 会识别每张图在详情页中的角色，并使用对应策略：

| 角色 | 文案策略 | 示例 |
|------|----------|------|
| 首屏/钩子 | 一句话核心利益点，≤15字 | "7天焕发好气色" |
| 痛点共鸣 | "你是否..."句式引发共鸣 | "你是否经常头晕乏力？" |
| 解决方案 | 产品如何解决痛点 | "3倍铁吸收，从根源改善" |
| 功效说明 | 数据化表达 | "14天见效，30天巩固" |
| 信任背书 | 权威性展示 | "国食健字号认证" |
| 对比优势 | 具体差异点 | "螯合铁 vs 普通铁，吸收率提升3倍" |
| 使用场景 | 人群+场景 | "孕期每日必备" |
| 用户证言 | 好评展示 | "坚持一个月，气色明显变好" |

### 避免文案重复

- 每张图只输出适合其角色的文案
- 不堆砌所有卖点
- AI 会理解上下文，避免相邻图片文案雷同

### 卖点扩散思维

基于产品的固定属性（功能、原料），AI 会扩散出更多角度：
- 功能 → 用户可感知的好处
- 原料 → 权威背书 + 差异化优势
- 人群 → 具体使用场景

## 输出规范

生成的 Word 工单遵循以下格式：

1. **标题**：`YYYYMMDD-电商-产品名称-负责人`
2. **规格信息**：
   - 图片类别：详情页(需要PSD)+主图
   - 尺寸：详情页 1440 宽，主图 1440×1440 + 1440×1920
   - 数量：详情页X屏 + 主图X张×2
3. **内容**：先主图，后详情页
4. **每条记录**：图片（200px宽度） + 对应文案

## 常见问题

**Q: API 报错 429 限流怎么办？**
A: 程序有自动重试机制，稍等即可。如频繁报错，可调大 `API_DELAY` 值。

**Q: 文案和竞品差异太大？**
A: 检查卖点文件是否涵盖了足够的产品信息，特别是与竞品相似的功效描述。

**Q: 生成的文案重复？**
A: 这是 AI 没有正确识别图片角色的问题，可尝试重新运行或调整卖点描述。
