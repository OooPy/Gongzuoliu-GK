# 落地页工单制作工作流

## 使用前提

首先阅读 Skill 文档了解完整流程：

```
查看 d:\Antigravity\测试文件夹\20260206 落地页工单制作\SKILL.md
```

## Step 1: 询问产品信息

**开始任何产品的落地页制作前，必须询问用户以下信息**：

1. **产品名称**：例如"雪叶红口服液"
2. **文案方向**：例如"孕妇补铁"、"气血虚补血"、"宫寒痛经"
3. **产品固定属性**（不可更改的重点）：
   - 功能性：如"补血养血"、"改善睡眠"
   - 原料成分：如"阿胶"、"红参"、"铁元素"
   - 权威背书：如"国食健字号"、"百年老字号"
   - 目标人群：如"孕妇"、"职场女性"
4. **竞品素材位置**：包含主图和详情页切图的文件夹路径

## Step 2: 准备工作

1. 确认竞品素材文件夹结构：

```
竞品文件夹/
├── 主图/           # 主图图片
└── 详情页（切图）/  # 已切好的详情页图片
```

1. 根据用户提供的产品信息，创建卖点文件 `产品名称卖点.txt`

2. 确保 `.env` 文件已配置 `GOOGLE_API_KEY`

## Step 3: 修改配置

编辑 `scripts/main.py` 中的配置区域：

- COMPETITOR_FOLDER：竞品素材文件夹名称
- PRODUCT_DOC：产品卖点文档名称
- PRODUCT_NAME：产品名称
- COPY_DIRECTION：文案方向
- AUTHOR_NAME：负责人

## Step 4: 执行生成

// turbo

```powershell
cd d:\Antigravity\测试文件夹\20260206 落地页工单制作
python scripts/main.py
```

## Step 5: 查看结果

生成的工单在 `output/` 目录下，格式为 `YYYYMMDD-电商-产品名-负责人.docx`

## 关键原则

1. **避免文案重复**：AI 会识别每张图的角色，只输出适合该角色的文案
2. **卖点扩散思维**：基于产品固定属性（功能、原料），扩散出更多用户可感知的好处
3. **电商专家思维**：10年电商运营专家的视角，理解转化漏斗
