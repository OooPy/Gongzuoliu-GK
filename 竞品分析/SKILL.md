# 医药保健品竞品深度分析 Skill

## 功能概述

基于 Gemini AI 视觉能力 + 结构化数据分析，以医药保健品行业大师级视角，对竞品进行 9 大维度深度分析。

**核心能力**：
- 🎨 视觉识别竞品主图、详情页截图，分析卖点策略与设计思路
- 📊 读取 Excel 运营数据（销售/SKU/搜索词/流量等），深度解读
- 🔍 9大维度专业分析：主图策略、详情页逻辑、规格设计、活动促销、评价洞察、销售数据、SKU分析、搜索词、流量来源
- 💡 自动生成综合竞品评估与差异化策略建议
- 📄 输出专业 Word 分析报告（保存在竞品文件夹内）

**分析风格**：
- 医药保健品行业15年+电商运营大师级专家视角
- 首次分析达到80分水平，支持对话式迭代优化
- 结合行业合规要求、消费者心理、竞争格局做专业解读

**项目代码位置**：`竞品分析/scripts/`

## 使用流程

### Step 1: 环境准备

首次使用时安装依赖：

```powershell
cd 竞品分析
pip install -r requirements.txt
```

### Step 2: 配置 API Key

在 `竞品分析/` 目录下创建 `.env` 文件：

```
GOOGLE_API_KEY=你的API密钥
```

### Step 3: 准备竞品素材

每个竞品对应一个文件夹，支持以下结构：

```
竞品素材/
├── 竞品A_品牌名/
│   ├── 主图+详情页/       # 主图和详情页截图
│   │   ├── 主图_1.jpg
│   │   ├── 主图_2.jpg
│   │   └── 详情长图_1.jpg
│   ├── 详细信息.xlsx      # 运营数据（销售/SKU/搜索词/流量等）
│   ├── 链接内首屏信息.png  # 可选：首屏截图
│   ├── 支付人群画像.pdf   # 可选：人群画像
│   └── 补充信息.txt       # 可选：手动补充的文字信息
├── 竞品B_品牌名/
│   └── ...
```

**支持的数据来源**：
- 图片：jpg/png/webp（主图、详情页、截图等）
- Excel：运营数据（支持多Sheet，自动识别内容）
- PDF：支付人群画像等（本地提取文本，不消耗API）
- 文本：补充信息

**注意**：
- 不需要每个类型都有，有什么数据放什么
- 主图/详情页可以混合放在一个文件夹，会按文件名自动分类
- Excel 数据越完整，分析越深入

### Step 4: 修改配置

编辑 `竞品分析/scripts/main.py` 中的配置区域：

```python
# 竞品素材根目录
COMPETITORS_BASE_FOLDER = r"d:\竞品信息"

# 我方产品信息（可选，用于差异化建议）
OUR_PRODUCT_INFO = ""

# 电商平台
PLATFORM = "天猫"

# API 调用间隔（秒）
API_DELAY = 1.0
```

### Step 5: 执行分析

```powershell
cd 竞品分析
python scripts/main.py
```

程序会：
1. 扫描竞品文件夹中的图片和数据文件
2. 读取 Excel 运营数据
3. AI 视觉分析主图策略和详情页逻辑
4. AI 数据分析（规格/活动/评价/销售/SKU/搜索词/流量）
5. 生成综合竞品评估
6. 输出 Word 报告到竞品文件夹内

### Step 6: 迭代优化

首次分析约80分水平，可以继续对话：
- 「主图策略部分再详细分析一下」
- 「评价洞察再深入挖掘」
- 「给出更具体的差异化方案」

## 9大分析维度

| 维度 | 数据来源 | 分析内容 |
|------|----------|----------|
| 主图卖点与策略 | 主图截图 | 逐张主图分析：角色定位、卖点、文案策略、视觉设计、合规评估 |
| 详情页逻辑 | 详情页截图 | 内容逻辑线、说服路径、信任构建、成分表达、差异化 |
| 规格设计 | Excel 数据 | SKU规格、定价策略、引流款/利润款分析 |
| 活动促销 | Excel/截图 | 参与活动、优惠券策略、促销力度 |
| 用户评价 | Excel/截图 | 评价量级、好评差评关键词、消费者洞察 |
| 30日销售 | Excel 数据 | 销售额/量、日均、趋势、品类定位 |
| SKU销售 | Excel 数据 | 各SKU占比、热销款、策略洞察 |
| 搜索词转化 | Excel 数据 | Top搜索词、转化率、品牌词vs品类词 |
| 流量来源 | Excel 数据 | 渠道分布、付费/免费比例、健康度 |

## 常见问题

**Q: 可以只分析一个竞品吗？**
A: 可以。`COMPETITORS_BASE_FOLDER` 直接指向单个竞品文件夹即可。

**Q: 不想分每个子文件夹怎么办？**
A: 直接把所有截图扔在竞品文件夹下，程序会做综合分析。

**Q: API 报错 429 限流？**
A: 调大 `API_DELAY` 值（建议 2-3 秒）。

**Q: 图片太多会不会超限？**
A: 每个类别最多处理 10 张图，超过的会自动截断。
